==Quick Start==
cd /var/www
git clone https://github.com/andy-s-clark/magento.git

1) Edit config.sh as needed.
2) Edit config.sh and setup.sh if a different base directory is needed
3) setup/setup.sh doit


== Directory structure ==
modules (modules)
patches (patches for Magento 1.x)
readme.txt (this file)
setup (setup script)

src (downloaded magento source and sample data)
magento-1.7.0.2 (extracted magento source with symbolic links to modules)
magento (symbolic link to magento-1.x.x.x)
